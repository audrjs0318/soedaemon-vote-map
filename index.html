공유하기

이 페이지를 웹에 공유하는 방법을 선택하세요.

방법 1 가장 쉬움 Netlify Drop

  1. https://app.netlify.com/drop 접속
  2. "vote map" 폴더 전체를 페이지에 드래그&드롭
  3. 즉시 URL 생성 (예: https://amazing-site-abc123.netlify.app)
  4. 생성된 URL을 복사해서 공유

방법 2 무료 GitHub Pages

  1. https://github.com에서 계정 만들기
  2. New Repository → 이름: vote-map, Public 선택
  3. index.html 파일 업로드 (Add file → Upload files)
  4. Settings → Pages → Source: main branch / (root) → Save
  5. URL: https://[사용자명].github.io/vote-map

방법 3 고급 Vercel

  1. https://vercel.com → Continue with GitHub
  2. "vote map" 폴더를 드래그&드롭
  3. Deploy 클릭 → 즉시 URL 생성
